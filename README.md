# Stunde Null – Interaktive Lernlandschaft

Dieses Repo enthält eine statische Lernlandschaft (HTML/CSS/JS) mit vier Video-Stationen und zwei Quiz-Bereichen mit Sofortkorrektur.

## Struktur
- `index.html` – Layout und Inhalte
- `styles.css` – Gestaltung und Hintergrund (inkl. halbtransparenter GIF-Ebene)
- `app.js` – Quiz-Logik und Fragenkatalog
- `assets/` – Medien (hier liegt die Hintergrund-GIF)

## GIF-Hintergrund
Lege die Datei `StundeNull.gif` in `assets/` ab. Sie wird automatisch als halbtransparenter Hintergrund eingeblendet.

## Hinweise zu den Videos
Die Videos werden direkt aus Dropbox-Links geladen (`raw=1`). Falls ein Video nicht lädt, prüfe die Freigabeeinstellungen oder ersetze den Link.

## Start
Einfach `index.html` im Browser öffnen.
